# Chat Features Implementation

## Серверная часть (NestJS)

### Новые схемы
- **ChatSettings** (`src/schemas/chat-settings.schema.ts`)
  - `chat_id` - ID чата
  - `identity_id` - ID пользователя
  - `is_pinned` - закреплен ли чат
  - `is_muted` - отключены ли уведомления
  - `unread_count` - количество непрочитанных сообщений
  - `last_read_at` - время последнего прочтения

### Новые сервисы
- **ChatSettingsService** (`src/chat/chat-settings.service.ts`)
  - `togglePin()` - переключить закрепление
  - `toggleMute()` - переключить мут
  - `incrementUnread()` - увеличить счетчик непрочитанных
  - `markAsRead()` - пометить как прочитанное
  - `getSettings()` - получить настройки чата
  - `getAllUserSettings()` - получить все настройки пользователя

### Новые API эндпоинты
- `PATCH /chat/:chat_id/pin` - закрепить/открепить чат
- `PATCH /chat/:chat_id/mute` - включить/выключить звук
- `PATCH /chat/:chat_id/read` - пометить чат как прочитанный

### WebSocket события
- `toggle_pin` - переключить закрепление (реалтайм)
- `toggle_mute` - переключить мут (реалтайм)
- `mark_read` - пометить как прочитанное (реалтайм)
- `unread_update` - обновление счетчика непрочитанных
- `pin_update` - обновление статуса закрепления
- `mute_update` - обновление статуса мута

## Клиентская часть (Swift)

### Новые модели
- **ChatListItem** обновлена:
  - `isPinned: Bool` - закреплен ли чат
  - `unreadCount: Int` - количество непрочитанных
  - `isMuted: Bool` - отключены ли уведомления

### Новые сервисы
- **WebSocketManager** (`Services/WebSocketManager.swift`)
  - Подключение к WebSocket серверу
  - Обработка реалтайм обновлений
  - Callbacks для обновлений (unread, pin, mute)

- **NotificationManager** (`Services/NotificationManager.swift`)
  - Запрос разрешений на уведомления
  - Отправка локальных уведомлений
  - Управление badge счетчиком

### Новые API методы
- `toggleChatPin()` - закрепить/открепить чат
- `toggleChatMute()` - включить/выключить звук
- `markChatAsRead()` - пометить как прочитанное

### UI обновления
- **ChatsListView**:
  - Закрепленные чаты отображаются вверху с иконкой 📌
  - Синий фон для закрепленных чатов
  - Зеленый badge с количеством непрочитанных
  - Иконка 🔕 для чатов без звука
  - Swipe actions для pin/mute
  - Автоматическая сортировка (закрепленные → по времени)

- **ChatListRow**:
  - Badge с количеством непрочитанных на аватаре
  - Иконка закрепления
  - Иконка мута
  - Жирный текст для непрочитанных сообщений

## Реалтайм обновления

### Поток данных
1. Пользователь отправляет сообщение
2. Сервер увеличивает `unread_count` для других участников
3. WebSocket отправляет `unread_update` всем участникам
4. Клиент обновляет UI и badge счетчик
5. При входе в чат отправляется `mark_read`
6. Счетчик обнуляется для всех устройств пользователя

### Уведомления
- Локальные уведомления при новых сообщениях
- Badge на иконке приложения с общим количеством непрочитанных
- Уведомления не приходят для замьюченных чатов
- Автоматическое обновление badge при изменениях
