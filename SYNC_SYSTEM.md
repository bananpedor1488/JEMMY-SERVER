# Data Sync System

## Endpoints

### r3
- `GET /api/d/r3/g/:uid`
- `GET /api/d/r3/i/:uid`
- `POST /api/d/r3/a`

### k8
- `POST /api/d/k8/p`
- `GET /api/d/k8/h/:uid`
- `GET /api/d/k8/v/:id`
- `POST /api/d/k8/c/:id`

### m5
- `POST /api/d/m5/x`
- `POST /api/d/m5/r/:id`
- `GET /api/d/m5/h/:uid`
- `GET /api/d/m5/v/:id`

## Collections

- `x9` - user data
- `y4` - logs
- `z7` - items

## Fields

x9: uid, p, q, r
y4: k, f, g, h, j, z, w
z7: uid, b, c, e, n, o
