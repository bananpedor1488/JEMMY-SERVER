# Jemmy - Новые фичи

## Бэкенд API

### Поиск по тегу
`GET /api/identity/search/:tag`
- Поиск пользователя по уникальному тегу (например: `ABC123`)

### Одноразовые ссылки
`POST /api/identity/generate-link`
- Генерирует одноразовую ссылку типа `jemmy://chat/TOKEN`
- Ссылка действует 24 часа

`POST /api/identity/use-link`
- Использует одноразовую ссылку для создания чата
- Ссылка становится недействительной после использования

### Кастомизация профиля
`POST /api/identity/update`
- Обновление username, bio, avatar_seed

## Модель Identity
Добавлены поля:
- `tag` - уникальный 6-символьный тег (например: `ABC123`)
- `bio` - описание профиля
